Classifying Blood Clot Origin using DeepLearning

This research project addresses the critical challenge of classifying blood clot origins in ischemic stroke patients using deep learning techniques. By analyzing histopathological images from the Mayo Clinic STRIP AI dataset, we developed a system that can accurately distinguish between cardioembolic (CE) and large artery atherosclerosis (LAA) origins. The project combines advanced preprocessing techniques with state-of-the-art deep learning models to achieve high accuracy while maintaining computational efficiency.
